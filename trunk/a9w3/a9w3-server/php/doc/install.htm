Local VHost Config ^^^^^^^^^^^^^^^^^^^^^ 1. install apache 2.0.x 2. unzip apache2.zip to $TEMP_DIR 3. copy $TEMP_DIR/mod_bw/Release/mod_bw.so to $APACHE2_HOME/modules/ 4. append $TEMP_DIR/conf/httpd.conf's content to $APACHE2_HOME/conf/httpd.conf (edit a9w3 home according to yourself) 5. restart apache2(httpd) 6. visit http://127.0.0.8(with bw) and http://127.0.0.9 2.
